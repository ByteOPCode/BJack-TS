#Usage 

With nodejs 14+ and npm 7+ 
   ` npm install` 
   ` npm start` 
and 
   ` npm test`

With Docker
    `docker-compose -f docker-compose.yml up`
                    (or)
    `docker build -t BlackJack:latest .`
    `docker run BlackJack`
